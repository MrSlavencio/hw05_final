# Социальная сеть **YATUBE**


## Описание проекта

Социальная сеть позволяет зарегистрированным пользователям публиковать посты с группами и без, прикреплять изображения к постам, комментировать посты, подписываться на авторов поста. Незарегистрированные пользователи могут только просматривать записи.<br/><br/>
Данный код предназначен для взаимодействия через браузеры.

## Как развернуть проект

Клонировать репозиторий и перейти в него в командной строке:

```
git clone https://github.com/MrSlavencio/hw05_final.git
```

```
cd api_final_yatube
```

Cоздать и активировать виртуальное окружение:

```
python3 -m venv venv
```

```
source venv/bin/activate
```

```
python3 -m pip install --upgrade pip
```

Установить зависимости из файла requirements.txt:

```
pip install -r requirements.txt
```

Выполнить миграции:

```
python3 manage.py migrate
```

Запустить проект:

```
python3 manage.py runserver
```
